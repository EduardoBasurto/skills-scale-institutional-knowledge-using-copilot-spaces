# OctoAcme Project Management Docs

Welcome! This README provides an overview of the project management practices used at OctoAcme and quick links to all process documents.

## About OctoAcme Project Management

We use a structured, iterative approach to managing projects, focusing on:

- Clear roles and responsibilities
- Customer-first delivery and measurable outcomes
- Iterative planning, delivery, and review cycles
- Transparent risk management and open communication
- Continuous improvement through retrospectives

Learn more about each phase and artifact in the documents below.
